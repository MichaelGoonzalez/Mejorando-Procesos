Pasos para construir este proyecto:

1. Crear los archivos: index.html, style.css, y una carpeta para guardar los estilos y scripts.
2. Añadir Bootstrap al proyecto (usar CDN es más sencillo).
3. Diseñar la estructura HTML para el formulario de inicio de sesión.
4. Personalizar el diseño con un archivo CSS (style.css).
5. Explicar cómo funcionan las clases de Bootstrap (como container, row, col, btn).
6. Probar el formulario en el navegador y hacer ajustes.
